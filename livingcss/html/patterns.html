Entries

Entries

Base style and markup for blog/post/entry. .entry__item and .entry__card are used for listing.
.entry__content is used for single view.

Entry Title

An entry title base style. Used by default in single view.
Redefined in .entry__item and .entry__card.

A title

 
<h1 class="entry__title">A title</h1>

Entry content

.entry__content is the container class for WordPress wysiwyg. link are underline by default in this container. Link are underlined by default. Unordered list are displayed with standard disc. Images alignements are managed by WordPress using .alignleft, .alignright and .aligncenter classes. see more details at components/_img.scss Galleries are managed by WordPress using .gallery class. see more details at components/_img.scss
